# 隐私与平台安全说明

这个工具建议只作为“本地筛选看板”使用：手动复制候选人文本，粘贴到本地页面，再在本机完成筛选和备注。

## 不要提交到公开仓库

- API key、`.env`
- Chrome 登录状态、浏览器 Profile
- BOSS cookie、storage state、二维码、登录截图
- 候选人数据、简历截图、聊天记录
- 微信、电话、邮箱等联系方式
- `candidates_db.json`、`cookies/`、`chrome-profile-*`、`logs/`、`backups/`

## 平台风险

自动采集、批量访问、自动点击、自动打招呼、批量提取未读、书签脚本扫描页面，都可能被招聘平台识别为异常行为，并触发账号或 IP 限制。

建议：

- 只手动复制粘贴候选人内容
- 不做自动采集和批量访问
- 不绕过安全验证
- 不在访问受限期间继续刷新或重试
- 联系候选人时保持人工判断和人工操作

## 数据合规

候选人信息属于敏感个人信息。请只在合法、必要、授权的招聘场景下使用，并控制访问权限。
