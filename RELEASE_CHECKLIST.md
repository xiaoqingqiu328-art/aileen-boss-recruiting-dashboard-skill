# 发布前检查清单

上传 GitHub 或发给别人前，请确认：

- [ ] 没有 API key、`.env`、模型密钥
- [ ] 没有 Chrome Profile、cookie、storage state
- [ ] 没有真实候选人数据、简历截图、聊天记录
- [ ] 没有微信、电话、邮箱等联系方式
- [ ] 没有 `candidates_db.json`
- [ ] 没有 `cookies/`
- [ ] 没有 `chrome-profile-*`
- [ ] 没有 `logs/`
- [ ] 没有 `backups/`
- [ ] README 已说明平台风险：不建议自动采集、批量访问、自动打招呼或绕过验证

建议公开发布时只保留程序文件、示例配置和说明文档。
